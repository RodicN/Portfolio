La Paillote

Site vitrine moderne et responsive pour un restaurant gastronomique, alliant design élégant et expérience utilisateur intuitive

2024
Site Vitrine
4 semaines

Vue d'ensemble

La Paillote est un restaurant spécialisé dans la cuisine asiatiques qui souhaitait renouveler sa présence digitale. Le site vitrine devait refléter l'identité du restaurant : élégance, authenticité et modernité.

Le projet visait à créer une expérience web immersive permettant aux visiteurs de découvrir l'atmosphère unique du restaurant, consulter le menu en temps réel.

Problématique & Objectifs

Problématique :

L'ancien site était obsolète, non responsive et ne permettait pas de mise à jour facile du menu. Le restaurant perdait des clients potentiels faute d'une présence web attractive et fonctionnelle.

Objectifs clés :

Responsive Design

Garantir une expérience optimale sur tous les appareils

Menu Dynamique

Permettre une mise à jour facile du menu par le restaurateur

Stack Technique

HTML5
CSS3
JavaScript
PHP
MySQL

Architecture MVC avec séparation claire des responsabilités. Interface frontend responsive utilisant CSS Grid et Flexbox. Backend PHP pour la gestion du menu et du système de réservation, avec stockage MySQL.

Mon Rôle & Livrables

Rôle : Développeur Fullstack

Responsabilités :

  • Conception et développement de l'interface utilisateur responsive
  • Développement du backend PHP et de la base de données MySQL
